# 💬⇄💬 双方向プログラミングスタジオ

中学校 技術・家庭科(技術分野)「D 情報の技術」—
**ネットワークを利用した双方向性のあるコンテンツのプログラミング** のための自作Web教材。

ブラウザだけで動きます(生徒のインストール・アカウント登録は不要)。

## 生徒ができること

1. **① しくみを知る** — 一方向/双方向のコンテンツの違いをアニメーションで理解
2. **② チャット体験** — 完成品のチャットで、本物のネットワーク通信を体験
3. **③ プログラミング** — ブロック(Blockly・日本語)で双方向コンテンツを自作
   - ミッション1: あいさつを送ろう(順次処理)
   - ミッション2: 自動で返事をしよう(イベント)
   - ミッション3: ことばに反応するボット(条件分岐)
   - ミッション4: おみくじボット(乱数)
   - ミッション5: じゃんけん対戦!(総合・ゲーム)
   - 発展: 生成されたJavaScriptの表示・改造モード
4. **④ 先生用** — 部屋のモニター/ログ書き出し/全消去(簡易パスコード保護つき)

## セットアップ(先生向け)

**[docs/teacher-guide.md](docs/teacher-guide.md) を参照してください。**

- Firebase(無料)を設定するとクラス全員での本物の通信ができます(約10分・一度だけ)
- 未設定でも「練習モード」(端末内でボット相手)として動作します

## 技術構成

- 静的サイト(GitHub Pagesで動作)/ HTML + CSS + JavaScript のみ
- [Blockly](https://github.com/google/blockly) v11(`lib/` に同梱・日本語化済み)
- 通信: Firebase Realtime Database(設定時のみ・CDN読み込み)
- 送信レート制限つき実行エンジン(ボット同士の無限ループは自動停止)
